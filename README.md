# Nori UI PureScript

[![Build Status](https://travis-ci.org/shybyte/nori-ui-purescript.svg?branch=master)](https://travis-ci.org/shybyte/nori-ui-purescript)

Nori will sing for you.

## Installation
Install nodejs, npm, bower, purescript, pulp and gulp.

    npm install
    bower install

## Run it
    gulp

## Test it
    scripts/test.sh
